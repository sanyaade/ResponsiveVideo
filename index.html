How to Use

Add video element to your html inside a container div that is styled to scale with the size of the browser window.

<div id="container">
  <video id="myVideo" width="640" height="360" poster="img/poster.jpg" controls autoplay>
    <source src="vid/video.mp4" type="video/mp4"  />
  </video>
</div>

First, link to jQuery and then to responsiveVideo.js. Next, instantiate the plugin and apply any custom settings to it. Last, use the plugin to target the video element(s) on the page.

<script src="http://code.jquery.com/jquery-1.7.min.js"></script>
<script src="js/responsivevideo.js"></script>
<script>
  var responsiveVideo = new $.responsiveVideo({
    player: 'swf/vidobject.swf',
    buttonColor: '#FFFFFF',
    accentColor: '#BBBBBB',
    backgroundColor: '#111111',
    scaleVideo: true,
    fullScreen: true
  });
  responsiveVideo.embed("myVideo");
</script>

Parameters are:

  • player: path to vidobject.swf
  • buttonColor: controls button color (default: #ffffff)
  • accentColor: controls accent color (default #bbbbbb)
  • backgroundColor: controls background color (default: #111111)
  • scaleVideo: set to false if you don't want the video to scale
  • fullScreen: fullScreen: set to false to disable full screen mode
